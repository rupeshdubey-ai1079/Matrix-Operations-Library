# Matrix-Operations-Library

A powerful Matrix Operations Library developed in C for performing various mathematical matrix computations efficiently.

## Features
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Determinant Calculation
- Scalar Multiplication
- Identity Matrix Generation
- Performance Benchmarking

## Technologies Used
- C Programming
- Dynamic Memory Allocation
- Modular Programming
- GCC Compiler

## Project Structure
```bash
.
├── main.c
├── matrix.c
├── matrix.h
└── README.md
```

## Compilation
```bash
gcc main.c matrix.c -o main -lm
```

## Run
```bash
./main
```
